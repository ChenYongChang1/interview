前端工程化
===

## 目的
* 预编译`SCSS`，生成`sourcemaps`
* 后处理`CSS`，比如`autoprefixer`，`base64`转码
* `JavaScript`代码`babel`处理，向下兼容
* 检测代码变化，实现浏览器热更新


## gulp
### 使用方式
引入插件——定义任务——执行任务
```
gulp.task(taskname, function(){
  	return gulp.src(filename)
  			   .pipe(plugin.methods)
  			   .pipe(gulp.dest(dest));
})
```

### 插件
* `gulp-sass`
* `gulp-sourcemaps`
* `gulp-autoprefixer`
* `gulp-livereload`
* `gulp-base64`
* `gulp-babel`
